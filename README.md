# Carmelé Studios – fotograafiaportfoolio

See projekt on eestikeelne ühe lehega fotograafiaportfoolio veebileht. Leht tutvustab fotograafi, kuvab portfooliot, sisaldab kontaktiosa ning kasutab kergeid animatsioone ja interaktiivset pildigaleriid.

## Projekti kirjeldus

Veebileht on loodud HTML-i, CSS-i ja JavaScriptiga. Kujundus on minimalistlik ja visuaalne, keskendudes fotograafi tööde esitlemisele. Lehel on fikseeritud navigatsioon, hero-sektsioon, tutvustus, portfoolio, broneerimise/kontakti osa ning jalus.

## Funktsionaalsused

- Responsiivne kujundus erinevatele ekraanisuurustele
- Fikseeritud navigatsioonimenüü
- Mobiilivaates avatav/suletav menüü
- Hero-sektsioon suure taustapildiga
- Fotograafi tutvustav „Minust“ sektsioon
- Portfoolio pildigalerii
- Hiire ja puutega lohistatav pildiriba
- ScrollReveal animatsioonid
- Kontaktivorm ja kontaktinfo
- „Tagasi üles“ nupp
- Sotsiaalmeedia link

## Kasutatud tehnoloogiad

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Remix Icon
- ScrollReveal
